这是博客的第一篇文章，用来确认站点已经搭建完成并可以正常发布。

目前已经完成

  • Hexo 本地环境已经配置好
  • GitHub Pages 部署已经打通
  • 当前博客主题为 ParticleX
  • 首页、导航、友情链接、邮箱复制等基础功能都已可用

接下来可以做什么

  1. 继续写你自己的第一篇正式文章
  2. 修改头像、背景图和首页文案
  3. 在友情链接页添加常看的博客
  4. 按需切换到 Butterfly 等其他主题

常用命令

npm run server
npm run build
npx hexo deploy

说明

这个博客目前托管在 GitHub Pages，上线后访问速度和页面更新会有少量缓存延迟，通常刷新或等待几分钟即可看到最新内容。

以后你写新文章、换主题、修改首页信息，基本都可以通过项目根目录里的交互工具完成。
