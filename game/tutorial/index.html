游戏の教程

这里放的是游戏项目的使用教程，不会进入博客文章列表。

项目一

序章·从粒子里出发

这个项目参考了抖音博主 不冷冰的抖音，然后我在他的基础上进行了细微的修改。

基础教程：需要打开笔记本的相机，然后按照下面的手势切换和触发效果。

在 Mode 1 中，五指张开：触发粒子生成。

Mode 1 还有一个小彩蛋：双手同时比剪刀手，也就是食指和中指伸出、无名指和小拇指弯曲，屏幕中间会出现 ZEZE。

这个彩蛋只在粒子特效模式里触发，五角星和爱心两个 Mode 1 都可以试。

双手食指靠近并保持：切换模式。

切换到 Mode 2：取景器之后，取景器里面是彩色的，其他区域是黑白的。

切换到 Mode 3：3D 牵引之后，可以伸出手掌生成小球。

然后可以用食指进行牵引。

螺旋丸！

然后这个教程就到此 结束啦。望各位玩的开心，٩( 'ω' )و 谢谢各位来玩！

Have fun
